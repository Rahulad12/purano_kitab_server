# Project Overview

## What is Purano Kitab?

Purano Kitab (Old Book) is a platform for buying and selling used books. Users can:
- List books for sale
- Browse and search books
- Favorite/bookmark books
- Manage their book inventory

## Target Users

- Students selling textbooks
- Book enthusiasts
- Second-hand book buyers

## Core Features

| Feature | Description |
|---------|-------------|
| User Registration | Email/password and Google OAuth |
| Book Listings | Create, read, delete book listings |
| Search | Search by title, author, category, price |
| Favorites | Save books to favorites |
| Categories | Organize books by category |
| Statistics | Track sales and favorites |

## Tech Stack

- **Backend**: NestJS 11
- **Database**: MongoDB
- **ORM**: Mongoose
- **Auth**: JWT + Google OAuth 2.0
- **API Docs**: Swagger (OpenAPI)
